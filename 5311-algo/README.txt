Running the project:
-------------------
After running the project we have 9 options: 
1. K-th Smallest element(Expected linear time)
2. K-th Smallest element(Worst case linear time
3. Top K elements
4. Classical Quick Sort
5. Randomized Quick Sort
6. Median Heuristic Quick Sort
7. Quick Sort with insertion Sort
8. Heap Sort
9. Exit

press 9 to exit from project. Otherwise press corresponding number to get corresponding value. 
we need to input the input file and the output file will be generated in output folder according to the input given like 1.txt/7.txt etc.
